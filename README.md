# Chat App Laravel

A real-time chat application backend developed using **Laravel** to support team communication, messaging, workspace collaboration, and activity management through a scalable RESTful architecture.

---

## 🚀 Overview

The Chat App Laravel project provides a secure and scalable backend for real-time communication. It includes authentication, team management, messaging APIs, and activity tracking while following Laravel's MVC architecture and RESTful API standards.

---

## ✨ Features

- User Authentication
- Team Management
- Messaging APIs
- Global Activity Tracking
- RESTful APIs
- Role-Based Authorization
- Database Management
- Error Handling
- Backend Architecture

## ⚙️ Installation

```bash
git clone https://github.com/ahsanali35103/chat-app-laravel.git

cd chat-app-laravel

composer install

cp .env.example .env

php artisan key:generate

php artisan migrate

php artisan serve
```
